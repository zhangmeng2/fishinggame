# 钓鱼网页游戏 - 设计文档

**日期**: 2026-06-13  
**状态**: 已确认

## 概述

一个基于浏览器的 2.5D 等距视角钓鱼游戏核心原型。玩家在岸边移动，用鼠标瞄准并向鱼群抛竿，钓到鱼得分。

## 技术选型

- **框架**: React + TypeScript + Canvas 2D
- **构建工具**: Vite
- **渲染**: 自定义等距投影，纯几何图形绘制（无精灵贴图）
- **视角**: 2.5D 等距（2:1 标准投影）
- **操作**: 键盘 WASD/方向键移动角色，鼠标瞄准 + 点击抛竿/收竿

## 数据流

每帧循环: update(deltaTime) → render(ctx)

### 输入 → 状态
- 键盘 WASD/方向键 → 更新 Player.position
- 鼠标移动 → 更新 Rod.angle（瞄准方向）
- 鼠标点击 → 抛竿（从瞄准态）或收竿（从等待态）

### 状态流转
```
待机 → 瞄准(鼠标控制方向) → 抛竿(浮漂飞向目标) → 等待(浮漂在水面)
                                                    ↓
                          收竿(再次点击) ← 上钩(鱼触碰浮漂)
```

### 上钩判定
鱼的碰撞圆与浮漂距离 < 阈值时判定上钩。收竿时上钩则加分，否则空竿。

## 等距渲染

### 坐标变换
```
sx = (x - y) * tileW / 2
sy = (x + y) * tileH / 2
```

### 场景布局（世界坐标）
- y=0..4: 草地/岸边（玩家移动区域）
- y=5..6: 浅水区（鱼不出现）
- y=7..14: 深水区（鱼生成区域）

### 渲染顺序
地面层 → 水面层 → 鱼 → 角色+鱼竿 → 钓线+浮漂

深度排序按世界坐标 y 值（y 越大越靠前绘制）。

### 视觉
- 地面: 灰绿色块
- 水面: 蓝色半透明层
- 角色: 圆角矩形
- 鱼竿: 从角色延伸的线段
- 浮漂: 红白相间小圆 + 水面波纹（同心圆）
- 鱼: 几何形状 + 颜色区分

## 玩法范围（核心原型）

- 1 个场景（湖边）
- 3 种鱼（不同分值、出现概率、移动速度）
- 得分系统（HUD 显示当前分数和最高分）
- 操作提示（HUD 底部文字提示当前可执行的操作）
- 无限游玩，无时间限制

## 不含范围

- 鱼竿升级、背包、图鉴
- 多场景切换
- 音效/音乐
- 移动端适配
- 存档/读档
- 鱼咬钩后的拉扯机制（点击即钓上）
